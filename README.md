> **Frontend Architecture & UI/UX:** Uma página de captura (Landing Page) moderna e responsiva desenvolvida como MVP para o **GPT Members**, um SaaS focado na gestão e orquestração de Agentes de Inteligência Artificial.

## 🎯 Sobre o Projeto

Este projeto foi desenvolvido como parte de um estágio acadêmico focado em engenharia de front-end. O objetivo principal foi traduzir os requisitos de negócios de uma plataforma de IA em uma interface de usuário atrativa, focada em conversão (captura de leads) e clareza de proposta de valor.

*Nota: Este repositório contém a interface estática (Frontend). O projeto encontra-se em fase de prototipagem e a integração com o backend de gestão de agentes não está pública.*

## ✨ Principais Desafios e Soluções

* **Design Focado em Conversão:** Estruturação de seções estratégicas (Hero, Features, Pricing e FAQ) para guiar o usuário até o Call to Action (CTA).
* **Performance:** Utilização do empacotador **Vite** em conjunto com **React** para garantir um carregamento quase instantâneo, vital para retenção de usuários em páginas de captura.
* **Componentização:** Arquitetura de pastas baseada em componentes reutilizáveis para facilitar a futura escalabilidade da aplicação quando o painel do SaaS for acoplado.
